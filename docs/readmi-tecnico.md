# Project Overview

TaskFlow is a full stack task management application developed using a modular frontend and backend architecture.

The project was designed to practice:
- REST APIs
- Express.js
- frontend/backend communication
- modular architecture
- middleware usage
- responsive design
- dark mode
- Git and GitHub workflows

---

# Technical Stack

## Frontend
- HTML5
- CSS3
- JavaScript
- Fetch API

## Backend
- Node.js
- Express.js
- Dotenv
- Cors
- Nodemon

---

# Application Architecture

The application is divided into two main parts:

## Frontend
Responsible for:
- rendering tasks
- user interaction
- filters
- search system
- dark mode
- responsive design
- API communication

## Backend
Responsible for:
- REST API
- request handling
- business logic
- middleware execution
- task management
