# Comparativa de asistentes IA

En este apartado se comparan distintos asistentes de inteligencia artificial aplicados al desarrollo web.

## Asistentes comparados

- ChatGPT
- Claude
- GitHub Copilot
- Cursor

## Objetivo

El objetivo es comprobar cómo ayudan estos asistentes a explicar conceptos técnicos, detectar errores, generar funciones y mejorar código existente.
