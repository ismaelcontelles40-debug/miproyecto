# Flujo de trabajo con Cursor

Cursor es un editor de código basado en inteligencia artificial que permite mejorar la productividad del desarrollo.

## Instalación y primeros pasos

Se ha instalado Cursor y se ha explorado su interfaz. Es similar a VS Code pero incluye herramientas de IA integradas.

## Funciones utilizadas

- Chat contextual para preguntar sobre el código
- Edición inline para modificar funciones directamente
- Composer para realizar cambios en varios archivos a la vez
- Revisión automática de código

## Ejemplos de uso

Se ha seleccionado una función del proyecto TaskFlow y se ha pedido a Cursor que la mejore.

También se ha utilizado para explicar partes del código y proponer mejoras en la estructura.

## Atajos de teclado

- Ctrl + K: edición inline
- Ctrl + L: abrir chat
- Ctrl + I: abrir Composer

## Mejora de productividad

Cursor permite escribir menos código manualmente y facilita la comprensión del proyecto. Es útil para refactorizar, detectar errores y generar funciones rápidamente.
