# Flujo de trabajo con Cursor

Cursor es un editor de código con funciones de inteligencia artificial integradas.

## Funciones probadas

- Chat contextual sobre el proyecto
- Edición inline de código
- Composer para modificar varios archivos
- Revisión de errores
- Mejora de productividad

## Atajos útiles

- Ctrl + K: edición inline
- Ctrl + L: abrir chat
- Ctrl + I: abrir Composer

## Ejemplo de uso

Se puede seleccionar una función de JavaScript y pedir a Cursor que la refactorice, añada comentarios o mejore su legibilidad.
